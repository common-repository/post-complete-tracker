Adds a "Complete" button after the post content for logged users.

== Description ==
Post Complete Tracker is a WordPress plugin that adds a "Complete" button after the post content for logged-in users. Once a user clicks on the button, the button text changes to "Completed". This plugin is useful for tracking the progress of users reading articles or posts on your website.

== Features ==
1. Easy to install and activate.
1. Adds a "Complete" button after the post content for logged-in users.
1. Once a user clicks on the button, the button text changes to "Completed"
1. Saves the completion status in the database
1. Compatible with the latest version of WordPress

== Installation ==
1. Install the plugin through the WordPress plugin directory or upload the plugin files to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. The "Complete" button will now appear after the post content for logged-in users

== Usage ==
1. Log in to your website as a user.
2. Navigate to enfo of a post on your website.
3. Click on the "Complete" button at the end of the post
4. The button text will change to "Completed" and the completion status will be saved in the database
